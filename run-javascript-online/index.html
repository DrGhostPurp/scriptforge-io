JavaScript code examples to try

// Fetch API – async/await

const res = await fetch('https://jsonplaceholder.typicode.com/posts/1');
const data = await res.json();
console.log(data.title);

// Import lodash from npm

import _ from 'https://esm.sh/lodash';
const arr = [1, 2, 3, 4, 5];
console.log(_.chunk(arr, 2));  // [[1,2],[3,4],[5]]

// Array methods – modern JS

const users = [
  { name: 'Alice', score: 92 },
  { name: 'Bob', score: 78 },
  { name: 'Carol', score: 95 }
];
const top = users
  .filter(u => u.score > 80)
  .sort((a, b) => b.score - a.score)
  .map(u => u.name);
console.log(top);  // ['Carol', 'Alice']

Frequently asked questions

How do I run JavaScript online?

Open ScriptForge, select JavaScript mode, type or paste your JS code, and click Run. Your code executes instantly in the browser with output shown in the console panel. No install or login needed.

Yes. ScriptForge supports npm package imports via esm.sh CDN. Use import statements to load lodash, axios, dayjs, React, and thousands of other packages directly in the browser.

Does ScriptForge support async/await?

Yes. ScriptForge fully supports modern JavaScript including async/await, Promises, ES6+ modules, arrow functions, destructuring, and all other modern syntax including top-level await.
