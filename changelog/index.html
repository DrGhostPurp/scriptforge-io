Changelog

Every feature, fix, and improvement — tracked publicly.

v0.5 Latest
March 2026

New Features

  • Real Python execution via Pyodide WebAssembly — runs in-browser, no server
  • npm package imports via esm.sh — import any package directly in browser JS
  • Live HTML preview panel — see rendered output instantly
  • GitHub Gist import — paste any Gist URL to load scripts instantly
  • Script export — download as .js, .py, .sh, or .html
  • Shareable URLs — scripts share via URL hash fragment
  • Keyboard shortcuts — Ctrl+Enter to run, Ctrl+S to save, Tab for indent
  • Status bar — shows line count, cursor position, and execution time

Improved

  • Dark/light theme toggle with localStorage persistence
  • Script sidebar with real-time search/filter
  • 10 built-in starter templates for Python, JS, HTML, and shell
  • Execution output panel with clear button and copy support
