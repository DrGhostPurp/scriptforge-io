What's inside

ScriptForge is built around one idea: your browser is already a fully capable development environment. We just unlocked it.

🐍
Real Python via Pyodide
Full CPython running in WebAssembly. NumPy, pandas, matplotlib, scikit-learn — all available. Not a subset. Not a transpiler. Actual Python.
📦
Any npm Package
Import any npm package via esm.sh with a single URL. No package.json, no node_modules, no bundler. The entire ecosystem, in your browser.
👁️
Live HTML Preview
Write HTML, CSS, and JavaScript and see the rendered output in a live panel. Build components and test layouts without leaving the tab.
💾
Local Storage Persistence
Scripts save to localStorage. They're there when you come back — no account, no sync, no cloud. Your scripts live in your browser.
🔗
GitHub Gist Import
Paste any GitHub Gist URL and import the script instantly. Share scripts with teammates by sharing a Gist link.
🔒
Private by Architecture
Your code never leaves the tab. No telemetry, no server execution, no data collection. Privacy isn't a policy — it's built into how it works.
⌨️
Keyboard-First
Ctrl+Enter to run. Ctrl+S to save. Tab for indent. Status bar shows line count, cursor position, and execution time.
📤
Export Anywhere
Download scripts as .js, .py, .sh, or .html. Share via URL hash. Import/export as JSON. Your work, your format.
